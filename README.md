# Currency Converter

A simple and fast currency converter application built with React and Vite.

## Features

- Convert between multiple currencies in real-time
- Responsive and user-friendly interface
- Fast build and hot-reload with Vite

## Getting Started

- use npm run dev command to run the application

### Prerequisites

- [Node.js](https://nodejs.org/) (v14 or higher recommended)
- [Yarn](https://yarnpkg.com/) (or use npm if preferred)

### Installation

- for installation use yarn i command

1. **Clone the repository:**
   ```bash
   git clone <https://github.com/lenah23/currency-change.git>
   cd <currency-change>
   ```
